Automated Particle Analysis via Electron Microscopy
  • Quantification of nanoscale precipitation in AA7050

    I learned and optimized two new automated data collection software in our microscopy center, together called the Automated Particle Workflow (APW) software package-- the first, Maps, allowed the automated collection of large tilesets of data on a sample; the second, Avizo2D, provided a GUI for building image processing recipes that could then be automatically applied to each tile. This was a new capability that enabled both collecting and analyzing large datasets overnight or over a weekend during otherwise unused instrument time.

    In this specific project, I collected a large tileset of images showing small particles (precipitates) embedded in multiple samples of an aluminum alloy and applied automated image processing, yielding datasets over one million points. I was then able to visualize the data using Python and the libraries pandas and matplotlib.
